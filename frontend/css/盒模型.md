## box-sizing:

- **content-box**（默认）

  - width/height 只包含内容
  - background-color 会填充到包括 border 以内.

- **border-box**（推荐）

  - width/height 包含, 内容 + padding + border
  
- **inherit**

  



## width/height

100%找父元素, 找到html, 就找window窗口大小




## 默认宽度

如果没有设置width, 有 margin, 实际 computed 后的 width == window - margin * 2, 所以 border/padding 不会影响width, 但是会影响 content 宽度

## margin 塌陷(历史问题, 决定给父元素好)

顶部/底部的边缘子元素, 会出现这个问题.

第一个子元素设置了top margin, 导致 top margin 作用在了 父元素.
