# @media

没有优先级, 相同标签写在后的会覆盖前边的



print: 打印机/打印预览

screen: 屏幕

all



# 特性

width (可以指的打印出来/screen)

```
@media(width:800px){

}
```

max-width (小于等于700)

min-width

device-width:设备屏幕宽度



# 运算符

max-width 可以和 min-width 连接



## and

```
@media(min-width:700px) and (max-width:800px){

}
```



## or(,也行)

```
@media(max-width:700px), (min-width:800px){

}
```



## not 

```
@media not screen {

}
```



## only(没什么用)



# 可写在外部引用

```
<link rel="stylesheet" media="screen and (min-width:1200px)" href="./css/huge.css">
```
