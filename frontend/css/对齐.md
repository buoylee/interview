# line-height(行高, 注意区分 height(容器高度))

font-size不变情况下,留白更多上下空白. 注意, 这个不是行与行直接的空隙, 他们是贴着的, 只是视觉上空着.



没有height时, height == line-height * 行数





## 单行文字垂直对齐

当 font-size < height时, line-height == height.



靠底部不科学, 只是一种解决方法
