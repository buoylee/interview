# this 是谁

执行到 包含 "箭头函数"的代码时, 才会确定 箭头函数 的 this.

执行到 setTimeout 时, 确定 this 为 vue(vm)



## 注意区分容易混淆的地方

外部定义(this == window), 后传入.

```
const arrow = () => console.log(this);

function outer() {
  setTimeout(arrow, 1000);
}
outer.call({ name: 'obj' });
```



产生不同的箭头函数(不同的this)

```
function makeArrow() {
  return () => console.log(this);
}

const a = makeArrow.call({ name: 'A' });
const b = makeArrow.call({ name: 'B' });

a(); // A
b(); // B
```
