> **目标**：掌握 ML 的核心概念和经典算法，理解「模型是怎么学习的」。这些概念是后续深度学习和大模型的根基。

## 🎯 本阶段核心思想

```
数据 → 特征 → 模型 → 损失函数 → 优化 → 评估 → 迭代

这个流程从传统 ML 到 LLM 都没变过，变的只是规模和自动化程度。
```
