# 01 - 主循环与状态机：逐行走读

> 本文用一个具体例子，跟踪 `Runner.run()` 从进入到返回的**完整数据流**。
> 每个关键步骤标注源码行号，并打印当前核心变量状态。

---

## 我们的例子

```python
from agents import Agent, Runner, function_tool

@function_tool
def get_weather(city: str) -> str:
    """查询城市天气"""
    return f"{city}今天晴，25°C"

agent = Agent(
    name="天气助手",
    instructions="你是一个天气查询助手",
    tools=[get_weather],
)

result = await Runner.run(agent, "北京天气怎么样？")
```

这个例子会经历**两轮循环**：
1. 第一轮：LLM 看到工具，返回 `tool_call(get_weather, {"city": "北京"})`
2. 第二轮：执行工具，结果喂回 LLM，LLM 返回最终文本

## 阶段一：入口 — Runner 到 AgentRunner

### 1.1 Runner.run() — 薄代理层

源码：`run.py:157-235`

```python
class Runner:
    @classmethod
    async def run(cls, starting_agent, input, *, context=None, max_turns=10, ...):
        runner = DEFAULT_AGENT_RUNNER   # run.py:222，全局单例 AgentRunner()
        return await runner.run(starting_agent, input, ...)
```

`Runner` 做的唯一一件事：转发到 `DEFAULT_AGENT_RUNNER`。

> **为什么这样设计？** 允许框架内部替换 runner 实现（实验性功能），而用户代码不变。

async def get_new_response(agent, system_prompt, input, ...):
    # 1. 可选的输入过滤
    filtered = await maybe_filter_model_input(...)
    # run_config.call_model_input_filter = None → 不过滤

    # 2. 获取模型和设置
    model = get_model(agent, run_config)
    # agent.model = None → 用默认模型（gpt-4.1）

    # tool_use_tracker 为空（第一轮没用过工具）→ 不重置

    # 4. LLM Start Hooks
    await asyncio.gather(
        hooks.on_llm_start(context_wrapper, agent, filtered.instructions, filtered.input),
        agent.hooks.on_llm_start(...) if agent.hooks else noop,
    )

    # 5. 调用 LLM（带重试）
    new_response = await get_response_with_retry(
        get_response=lambda: model.get_response(
            system_instructions="你是一个天气查询助手",
            input=[{"role": "user", "content": "北京天气怎么样？"}],
            model_settings=model_settings,
            tools=[FunctionTool(name="get_weather", ...)],
            output_schema=None,
            handoffs=[],
            tracing=...,
        ),
        retry_settings=model_settings.retry,
        ...
    )

    # 6. 累加 usage
    context_wrapper.usage.add(new_response.usage)

    # 7. LLM End Hooks
    await asyncio.gather(
        agent.hooks.on_llm_end(...) if agent.hooks else noop,
        hooks.on_llm_end(context_wrapper, agent, new_response),
    )

    return new_response
