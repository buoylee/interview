## 桶① 匹配什么字符

| 符号 | 含义 | 例 | 备注 |
|---|---|---|---|
| `abc` | 字面量 | `cat` 匹配 `cat` | |
| `.` | 任意字符(**默认不含换行**) | `a.c` 匹配 `abc` `a c` | 加 `re.S` 后含换行 |
| `\d` `\D` | 数字 / 非数字 | `\d+` | **默认含 Unicode 数字**,纯 ASCII 用 `re.A` 或 `[0-9]` |
| `\w` `\W` | 单词字符 `[a-zA-Z0-9_]` / 取反 | `\w+` | 默认含 Unicode 字母,如中文 |
| `\s` `\S` | 空白 / 非空白 | `\s+` | 含空格、`\t`、`\n` 等 |
| `[abc]` | 集合,三选一 | `[aeiou]` | 内部多数元字符失效:`[.]` 是字面点 |
| `[a-z]` | 范围 | `[0-9a-fA-F]` | `-` 放首尾或转义才是字面减号 |
| `[^abc]` | 取反集合 | `[^0-9]` 非数字 | `^` 只在**集合开头**才是取反 |
| `\.` `\*` … | 转义元字符成字面 | `\.` 字面点 | 见下方转义节 |

## 桶② 匹配多少个(量词)

| 符号 | 含义 | 贪婪默认 | 懒惰(+`?`) | 占有(+`+`,不回溯) |
|---|---|---|---|---|
| `*` | ≥0 个 | `a*` | `a*?` | `a*+` |
| `+` | ≥1 个 | `a+` | `a+?` | `a++` |
| `?` | 0 或 1 个 | `a?` | `a??` | `a?+` |
| `{n}` | 恰好 n 个 | `\d{4}` | — | — |
| `{n,}` | ≥n 个 | `\d{2,}` | `\d{2,}?` | `\d{2,}+` |
| `{n,m}` | n 到 m 个 | `\d{2,4}` | `\d{2,4}?` | `\d{2,4}+` |

> 贪婪/懒惰/占有的行为差异见 `01` 第 3–4 节。
> **占有量词 / 原子组 `(?>...)` 需 Python 3.11+**;Java 一直支持;Go 不回溯故不需要。

## 桶③ 匹配什么位置(零宽,不吃字符)

| 符号 | 含义 | 例 |
|---|---|---|
| `^` | 串首(`re.M` 下为**行首**) | `^From` |
| `$` | 串尾 / 末尾换行前(`re.M` 下为**行尾**) | `end$` |
| `\A` `\Z` | 严格串首 / 串尾(**不受 `re.M` 影响**) | |
| `\b` `\B` | 单词边界 / 非边界 | `\bcat\b` 不匹配 `category` |
| `(?=...)` | 前瞻肯定:右边**得是** | `\d(?=px)` 匹配 `9px` 的 `9` |
| `(?!...)` | 前瞻否定:右边**不能是** | `\d(?!px)` |
| `(?<=...)` | 后顾肯定:左边**得是**(Python **需定长**) | `(?<=\$)\d+` 取 `$100` 的 `100` |
| `(?<!...)` | 后顾否定:左边**不能是** | `(?<!\$)\d+` |

> Python 后顾**必须定长**:`(?<=ab)` 可,`(?<=a+)` 报错。要变长后顾装第三方 `regex` 模块。Go 无前瞻/后顾。

## 桶④ 分组与复用

| 符号 | 含义 | 备注 |
|---|---|---|
| `(...)` | 捕获组,占编号,可回引/单独取出 | `(\d{4})-(\d{2})` |
| `(?:...)` | 非捕获组,只为分组 | 不需要捕获时用它,略快、不占编号 |
| `(?P<name>...)` | 命名组(Python) | Java 是 `(?<name>...)` |
| `\1` `\2` | 反向引用:再来一个和第 n 组**相同**的 | `(\w)\1` 匹配叠字 `aa` |
| `(?P=name)` | 命名组反向引用(Python,模式内) | |
| `\|` | 或(二选一),**优先级极低** | `^(cat\|dog)$` 必须加组,否则成 `^cat` 或 `dog$` |

## 转义与特殊字符

- **元字符**(需 `\` 转义成字面):`. ^ $ * + ? { } [ ] ( ) | \ /`
- 字符集合 `[]` 内,只有 `^ - ] \` 需要留意,其余元字符自动变字面。
- `re.escape(s)`:把用户输入安全地拼进正则时,先转义。**拼接用户输入务必用它**,否则用户输入的 `.` `*` 会被当元字符。

```python
pat = re.compile(re.escape(user_input))   # 安全
```

## Flags(模式开关)

| 常量 | 简写 | 作用 |
|---|---|---|
| `re.IGNORECASE` | `re.I` | 忽略大小写 |
| `re.MULTILINE` | `re.M` | `^ $` 匹配**每行**首尾,而非整串 |
| `re.DOTALL` | `re.S` | `.` 也匹配换行 |
| `re.VERBOSE` | `re.X` | 忽略模式里的空白、允许 `#` 注释(写长正则用) |
| `re.ASCII` | `re.A` | `\d \w \s \b` 只认 ASCII(默认认 Unicode) |
| `re.UNICODE` | `re.U` | Python3 默认,通常无需写 |

**内联写法**(无需传 flags 参数):
- 全局:`(?i)abc`(放模式开头)
- 作用域:`(?i:abc)def`(只 `abc` 忽略大小写)、`(?-i:...)` 局部关闭(3.11+)

```python
re.search(r'cat', s, re.I)          # 等价
re.search(r'(?i)cat', s)            # 内联
```

## Python `re` API 速查

### 顶层函数

| 函数 | 返回 | 匹配范围 | 对应 Java | 对应 Go |
|---|---|---|---|---|
| `re.search(p, s)` | `Match` \| `None` | **任意位置**找第一个 | `matcher.find()` | `FindString` |
| `re.match(p, s)` | `Match` \| `None` | **从开头**起(不必到结尾) | `matcher.lookingAt()` | 用 `^` |
| `re.fullmatch(p, s)` | `Match` \| `None` | **整串**完全匹配 | `matcher.matches()` | `^...$` / `MatchString` |
| `re.findall(p, s)` | `list` | 所有匹配 | — | `FindAllString` |
| `re.finditer(p, s)` | 迭代器(`Match`) | 所有匹配(省内存) | — | `FindAllStringIndex` |
| `re.sub(p, repl, s)` | `str` | 替换全部 | `replaceAll` | `ReplaceAllString` |
| `re.subn(p, repl, s)` | `(str, n)` | 替换 + 次数 | — | — |
| `re.split(p, s)` | `list` | 按模式切 | `split` | `Split` |
| `re.compile(p, flags)` | `Pattern` | 预编译复用 | `Pattern.compile` | `MustCompile` |
| `re.escape(s)` | `str` | 转义字面量 | `Pattern.quote` | `QuoteMeta` |

> ⚠️ **`match` ≠ 整串**:`re.match(r'\d+', '123abc')` 成功(匹配 `123`)。要整串校验用 `fullmatch` 或加 `^...$`。这是 Java 转 Python 的高频坑——Java 的 `matches()` 是整串,Python 的 `match` 不是。

### Match 对象

| 方法 | 返回 |
|---|---|
| `m.group(0)` / `m[0]` | 整个匹配 |
| `m.group(1)` / `m[1]` | 第 1 个捕获组 |
| `m.group('name')` | 命名组 |
| `m.groups()` | 所有组的元组 |
| `m.groupdict()` | 命名组 `{名: 值}` |
| `m.start()` `m.end()` `m.span()` | 位置 `(start, end)` |

### `sub` 的替换串语法

```python
re.sub(r'(\d{4})-(\d{2})', r'\2/\1', '2026-06')      # -> '06/2026'  反向引用
re.sub(r'(?P<y>\d{4})', r'[\g<y>]', '2026')          # -> '[2026]'   命名引用 \g<name>
re.sub(r'\d+', lambda m: str(int(m[0])*2), 'a3b5')   # -> 'a6b10'    repl 传函数
```

`\g<0>` 是整个匹配;`repl` 传函数时,每个匹配调用一次,入参是 `Match`。

## 一段"全用上"的示例

```python
import re

text = "订单 ORD-2026-0042 金额 $1,299, 订单 ORD-2026-0099 金额 $58"
pat = re.compile(r"""
    ORD-(?P<year>\d{4})-(?P<seq>\d+)   # 订单号:年 + 流水(命名组)
    \s+金额\s+
    \$(?P<amount>\d{1,3}(?:,\d{3})*)   # 金额(千分位分组,不会误吃后面的分隔逗号)
""", re.VERBOSE)

for m in pat.finditer(text):
    print(m.group('year'), m.group('seq'), m.group('amount'))
# 2026 0042 1,299
# 2026 0099 58
```

`re.VERBOSE` 让你能换行 + 加注释写长正则——长正则一律这么写,可读性翻倍。下一章 `03` 是能直接抄的配方库 + 陷阱清单。
