## ??

**用于设置默认值**, 如果变量为 null, 取`??`后边的值, 非null则不变.

```
isFinish == true //有这行, isFinish 就是 true, 无则 false;
isFinish = isFinish ?? false
print(isFinish); 
```

```
isFinish ??= false // 简写
```
