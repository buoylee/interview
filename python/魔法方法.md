# 常见类别与代表

- 对象生命周期：__new__、__init__、__del__

- 字符串表示：__str__、__repr__

- 容器协议：__len__、__getitem__、__setitem__、__delitem__、__contains__

- 迭代协议：__iter__、__next__

- 上下文管理：__enter__、__exit__

- 调用/属性：__call__、__getattr__、__getattribute__

- 运算符重载：__add__、__sub__、__mul__…（算术）、__eq__、__lt__…（比较）、__hash__

- 描述符协议：__get__、__set__、__delete__

- 协程/异步：__await__、__aiter__、__anext__



上面那几类已经覆盖了日常开发 90% 的魔法方法。再补充一些常见但没提到的，你可以根据需要了解：

- 布尔/数值相关：__bool__、__int__、__float__

- 格式化/bytes：__format__、__bytes__

- 集合/哈希：__hash__

- 上下文管理衍生：__aenter__、__aexit__（异步 with）

- 异步迭代：__aiter__、__anext__

- 拦截设置/删除属性：__setattr__、__delattr__、__setitem__、__delitem__

- 描述符协议：__get__、__set__、__delete__

- Slots 优化：__slots__

- 容器推导支持：__reversed__

- 垃圾回收钩子：__del__

- 上下文管理器标识：__enter__ / __exit__ 之前已经提过

建议的学习策略还是那句：先掌握“核心”（对象构造、字符串表示、容器/迭代、上下文管理、运算符重载），遇到具体需求时再查官方文档就够了。官方参考：
