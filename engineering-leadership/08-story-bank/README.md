# 08 · 自我盤點 & 故事銀行

> 前面每一章你都填了一張故事卡。這章把它們**系統化成一個「故事銀行」**——一組精挑、打磨過、覆蓋全題型的故事,讓你面試當天**聽到任何行為題,都能從銀行裡掏出對應的那張**,而不是臨場硬擠。
>
> 這是整個 track 的彈藥庫。**面試成敗,八成在這一步做得紮不紮實。**

---

## 1. 為什麼要「銀行」?——不能臨場編

01 章說過:行為題追問會挖真假,編的故事追三層就崩。所以你**唯一**能依靠的,是**事先把真實經歷打磨成可講的故事、存好、練熟**。臨場才想「我有什麼例子」=必輸。

「銀行」這個比喻的關鍵:**你存的是錢(打磨好的故事),面試時是取款(快速調用),不是現場印鈔。**

---

## 2. 一張「好故事卡」長什麼樣

把前面各章的卡片格式統一成標準件:

```
【故事卡】
標題(一句話,自己一眼認出):________________________
─────────────────────────────────
S 情境:1-2 句背景(別超過)
T 任務:你要解決什麼 / 目標
A 行動:【我】做了 1、2、3(具體、有判斷,不用「我們」)  ← 最厚
R 結果:可感知 / 可量化的結果 + 外溢影響              ← 必有
─────────────────────────────────
主打維度:(00 章 5 維度,勾這故事最強的 1-2 條)
可答題類:(01 章 6 類,這張卡能 cover 哪幾類)
追問預備:最可能被追問的 1-2 個點 + 你的接法
一句話反思:如果再來一次,我會 ______(備反事實追問)
```

**質檢三條**(每張卡都過一遍):
1. **A 段夠厚、全是「我」嗎?**(沒有「我們…一下」)
2. **R 段有可感知結果嗎?**(數字最好,沒有也要有反饋/影響/避免的損失)
3. **標題能讓你一眼想起整個故事嗎?**

---

## 3. 覆蓋矩陣:用最少的卡,覆蓋最全的題

別貪多。目標是 **6-8 張卡覆蓋 01 章的 6 大題類**。關鍵技巧:**一張卡能 cover 多類**(一個衝突故事,既答「協作/衝突」也答「分歧/說服」)。

先畫這張矩陣,看哪類是空的(空的就是你要去挖的):

| 題類(01 章) | 在考的維度 | 我的故事卡 | 供料章 |
|---|---|---|---|
| 領導 / 影響 | Influence + Collaboration | ____________ | 02、06 |
| 協作 / 衝突 | Collaboration + Communication | ____________ | 07 |
| 當責 / 模糊 | Ownership + Judgment | ____________ | 03、04 |
| 規劃 / 優先級 | Judgment + Communication | ____________ | 03、04 |
| 失敗 / 學習 | Ownership + Maturity | ____________ | 本章 §5 |
| 分歧 / 說服 | Influence + Judgment | ____________ | 07、06 |

**目標**:每一行至少有一張卡;最好有一兩個「強故事」能橫跨 2-3 類。**有空行 = 那類題你會裸奔,優先去挖。**

---

## 4. 怎麼「挖」故事?——全題類自我盤點提問

對著下面的問題回憶,每想到一個就記下來(這些是原料,之後打磨成卡)。比 02 章更全:

**領導/影響**:誰因為你更快上手?你推動成過什麼沒人負責的事?
**協作/衝突**:你和誰意見不合 / 摩擦過?(PM 對齊就是,07 章)怎麼收的?
**當責/模糊**:哪次需求超模糊你自己理清了?哪次你扛下了一個不屬於你但會出事的坑?
**規劃/優先級**:哪次你把一個大專案拆得很漂亮?哪次同時一堆事你排對了優先級?
**失敗/學習**:你犯過什麼錯 / 哪個專案沒做好?(見 §5,必挖)
**分歧/說服**:你技術上說服過誰?用什麼說服的?

> **挖的時候別自我審查「這夠不夠厲害」。** 先全記下來,打磨階段再篩。很多「我覺得很小」的事,講法對了就是好故事。

---

## 5. 專挖「失敗 / 犯錯」這一類(最難備、最容易翻車)

「講一個你失敗/犯錯的事」是必考、且最多人答砸的題。為什麼難:你**本能想藏拙**,於是要嘛說「我想不到」(顯得沒反思/不誠實),要嘛說假失敗(「我太追求完美」這種凡爾賽,面試官一眼識破)。

**好的失敗故事,選材四標準**:
1. **真實**:真的搞砸過的事(假的會被追問拆穿)。
2. **低風險**:別選會嚇到面試官的災難(搞垮生產、害公司巨虧、人品問題)。選「判斷失誤/沒對齊/低估難度」這類專業失誤。
3. **有你的責任**:是「我當時 X 沒做好」,不是「都怪別人/環境」(甩鍋比失敗本身更扣分)。
4. **有學習 + 改進**:重點全在「我學到什麼 → 後來我怎麼改 → 同類問題不再犯」。**失敗題真正考的是『你會不會從失敗中長出能力』,不是失敗本身。**

> 你的現成素材:07 章「和 PM 反覆對不齊、做完才返工」——這本身就是一個絕佳失敗故事!失敗(對不齊返工)→ 反思(這是結構問題,責任在我建機制)→ 改進(對齊三件套)→ 結果(不再返工)。**一個故事,既是衝突題也是失敗題的彈藥。**

**失敗故事的 STAR 變體**:S 情境 → 我犯的錯(誠實)→ 我如何補救 → **我學到 + 我建立的機制(重頭戲)**。

---

## 6. 打磨:把「原料」變成「能講的卡」

挖出一堆原料後,挑 6-8 個最有料的,逐張打磨:
1. **套 STAR/SBI**(01 章),標出四段。
2. **加厚 A**:把「我們做了」改寫成「我做了 1、2、3」,顯出你的判斷。
3. **補強 R**:翻日誌/PR/監控找回數字;沒數字就寫反饋、影響、避免的損失。
4. **標維度 + 題類**:這卡主打哪 2 條軸、能答哪幾類題。
5. **預想追問**:面試官會挖哪裡?(套各章 B3)準備接法。
6. **念出來**:對著手機錄音念一遍,聽——S 是不是太長?A 是不是還在用「我們」?R 有沒有結果?**自己聽一遍,問題全暴露。**

---

## 你的 drill(本章核心產出)

**建起你的故事銀行**:
1. 填 §3 的覆蓋矩陣,標出空行。
2. 用 §4/§5 的提問,把空行補上原料。
3. 挑 6-8 個,按 §6 打磨成標準故事卡,存成一個文件(你的「銀行」)。
4. **必含一張失敗卡**(§5 標準)。
5. 每張卡錄音念一遍,按質檢三條(§2)自評。

> 這份文件就是你面試前唯一要反覆看的東西。**建好它,你就從「怕被問」變成「等著被問」。**

---

## 本章小結

- 行為面試靠的是**事先打磨好的真實故事**,不能臨場編——故事銀行就是你的彈藥庫。
- 好故事卡:**S/T 短、A 厚且全是「我」、R 有可感知結果**,標好維度+題類+追問預備。
- 用**覆蓋矩陣**確保 6 大題類無空行;**一張強卡可橫跨多類**,6-8 張足夠。
- **失敗題必備**:真實 + 低風險 + 有我責任 + 重在學習與改進;你的 PM 對齊故事就是現成的。
- 打磨關鍵:加厚 A、補強 R、**錄音念一遍**讓問題自己暴露。
