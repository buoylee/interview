# 01 · 行為面試的運作原理

> 上一章你拿到了地圖(5 維度)。這章拿工具:面試官**為什麼**問過去、**怎麼**問、怎麼**追問挖真假**,以及你回答要套的兩個框架(STAR / SBI)——講透原理,不是讓你背公式。
>
> 學完你會懂:同樣一段經歷,為什麼有人講完面試官猛點頭,有人講完面試官一臉「所以呢?」。

---

## 1. 為什麼面試官執著於問「過去」?

技術題能考你「會不會」,但考不出你「**在真實混亂裡會怎麼做**」。於是有了**行為面試(behavioral interview)**,它的全部理論基礎就一句話:

> **過去的行為,是預測未來行為最可靠的依據。**

所以面試官不問「如果遇到衝突你會怎麼辦」(這種假設題你可以隨便編一個漂亮答案),而是問「**講一個你真的遇到衝突的事**」。因為:
- 假設題考你的「**想法**」(嘴上會說);行為題考你的「**行為紀錄**」(真的做過)。
- 真實發生過的事,**追問就會露餡或發光**——編的故事追三層就崩,真的故事追十層還有細節。

**這對你的意義**:你不能靠臨場編。你要靠**事先把真實經歷整理成可講的故事**(這就是第 08 章故事銀行)。這章先教你「整理成什麼形狀」。

---

## 2. STAR:把一段經歷講成「有判斷、有結果」的故事

### 2.1 四段是什麼

STAR 是四個段落的縮寫,講任何「你做過的一件事」都套這個骨架:

| 段 | 全名 | 講什麼 | 一句話 |
|---|---|---|---|
| **S** | Situation 情境 | 背景:什麼專案、什麼處境 | 「舞台搭一下」 |
| **T** | Task 任務 | 你要解決的問題 / 你的目標 | 「我要幹嘛」 |
| **A** | Action 行動 | **你具體做了什麼**(重點) | 「我怎麼做的」 |
| **R** | Result 結果 | 產生了什麼可感知的結果 | 「結果怎樣」 |

### 2.2 大多數人講 STAR 死在哪(這才是重點)

知道四個字母沒用,90% 的人**比例失衡**:

- **S 講太長** ❌:鋪墊三分鐘背景,面試官還沒聽到你做了什麼就走神了。→ S 給 1-2 句夠了,只要聽得懂後面就行。
- **A 講太虛 + 用「我們」** ❌❌(頭號死罪):「我們團隊就一起討論,然後就解決了。」——面試官**完全分不出你個人貢獻了什麼**。行為面試的核心過濾器就是:**剝掉「我們」,只看「你」。** A 段必須是「**我**做了 1、**我**做了 2、**我**做了 3」。
- **R 沒有 / 不量化** ❌:故事講完沒下文,或只說「後來就好了」。→ R 要給**可感知的結果**,能有數字最好(返工降了、延遲從 X 到 Y、上線後零事故),沒數字也要給「對方反饋 / 被推廣 / 避免了什麼損失」。

> **黃金比例**:**S+T 是引子(20%),A+R 是肉(80%)。** A 講你的判斷和動作,R 證明它有用。你過去答不好,十有八九是 A 太虛、R 沒有——不是經歷不好,是**肉沒端出來**。

### 2.3 走一遍(同一件事,薄 vs 厚)

**事件**:線上一個介面偶發超時。

- **薄版**:「我們系統有個介面老超時,後來我們查了一下,優化了一下就好了。」(S 模糊、A 全是「我們…一下」、R = 「就好了」)→ 零訊號。
- **厚版**:
  - **S**:「我負責的訂單查詢介面,大促前偶發超時,P99 飆到 3 秒。」(1 句,夠了)
  - **T**:「我得在大促前把它壓回 500ms 內,不然會影響下單。」
  - **A**:「**我**先加了鏈路追蹤定位到是一個 N+1 查詢;**我**改成批量查並加了快取;**我**還順手給這條鏈路補了監控告警,免得以後悶聲復發。」(全是「我 + 具體動作 + 判斷」)
  - **R**:「P99 從 3 秒降到 200ms,大促當天零超時。那套監控後來也成了我們新介面的標配。」(數字 + 外溢影響)

厚版每一句都在餵 00 章的 5 維度(Ownership / Judgment / Communication…)。**這就是「翻譯」**。

## 3. SBI:講「人」的事(衝突 / 反饋)用這個

STAR 很適合講「**專案 / 成就**」。但講「**和人之間**」的事(衝突、給反饋、帶人),有個更貼的框架 **SBI**:

| 段 | 全名 | 講什麼 |
|---|---|---|
| **S** | Situation 情境 | 什麼場合 |
| **B** | Behavior 行為 | **對方做的具體行為**(只講事實,不貼標籤) |
| **I** | Impact 影響 | 那行為造成的**具體影響** |

**SBI 的精髓是「對事不對人」**(還記得 07 章嗎):
- ❌ 貼標籤:「他很不負責任。」(評判人 → 對方立刻防衛)
- ✅ SBI:「**昨天站會你說介面今天給(B),結果我等到下班沒收到、後面測試排期也順延了(I)**。」(只講行為 + 影響 → 對方無從反駁,只能解決)

**什麼時候用哪個**:講「我搞定了一個專案 / 難題」→ STAR;講「我和某人有摩擦 / 我給某人反饋 / 我帶某人」→ 故事外殼用 STAR,但**講到關鍵那句對話時用 SBI 的方式說**(你會顯得極其專業)。07 章好版逐字稿裡就藏了這個。

> SBI 不只是面試框架,**是你工作中給人反饋的標準動作**——下次想說「你怎麼又…」時,改成 S+B+I,衝突會少一半。

## 4. 面試官怎麼「追問」?——挖真假的四把鏟子

行為面試的殺傷力在**追問(follow-up)**。你的主答案只是開場,追問才見真章。提前知道他會往哪挖:

1. **往「你」挖**:你說「我們」,他就問「**那具體你做了哪部分?**」——他在剝離團隊、定位你個人貢獻。**對策**:主答案就把「我」說清楚,別等他挖。
2. **往「細節」挖**:「你說優化了,**具體怎麼做的?用了什麼?**」——真經歷有細節,編的會卡。**對策**:選你真的做過、能講出技術細節的故事。
3. **往「反事實」挖**:「**如果再來一次,你會怎麼做不同?**」——考你的反思力和成熟度。**對策**:每個故事預備一個「我學到 X / 下次我會 Y」。
4. **往「衝突點」挖**:「**有人反對你嗎?你怎麼處理?**」——考協作和影響力。**對策**:預備故事裡的分歧點和你怎麼推動的。

> **signal vs noise**:面試官整場在做一件事——**過濾掉聽起來好但分不清是不是你做的「noise」,留下能定位到你個人判斷與影響的「signal」。** 你的任務就是把每句話都變成 signal:多用「我」、給細節、給結果、給反思。

---

## 5. 行為題完整地圖(taxonomy):你會被問的全部,就這幾類

別怕「題目千變萬化」。行為題翻來覆去就這 6 大類,每類你只要備 1-2 個故事就夠覆蓋:

| 題類 | 典型問法 | 在考(5 維度) | 本 track 哪章供料 |
|---|---|---|---|
| **領導 / 影響** | 帶過人嗎?怎麼影響別人? | Influence + Collaboration | 02、06 |
| **協作 / 衝突** | 講個和同事 / PM 的衝突 | Collaboration + Communication | 07 |
| **當責 / 模糊** | 講個你扛下責任 / 需求很模糊的事 | Ownership + Judgment | 03、04 |
| **規劃 / 優先級** | 同時很多事你怎麼排?怎麼拆需求? | Judgment + Communication | 03、04 |
| **失敗 / 學習** | 講個你犯的錯 / 失敗的專案 | Ownership + Maturity | 08(專門挖) |
| **分歧 / 說服** | 你技術上和別人意見不合怎麼辦? | Influence + Judgment | 07 |

> **這就是為什麼故事銀行只要 6-8 張卡**:一張卡常能覆蓋多類(一個衝突故事,既answer「協作/衝突」也answer「分歧/說服」)。第 08 章教你用最少的卡覆蓋最全的題。

---

## 你的 drill

拿一個你最近的真實經歷(任何「我搞定了一件事」),做兩件事:
1. **標出 STAR 四段**:哪句是 S、T、A、R?
2. **體檢比例**:你的 A 段是「我 + 具體動作」還是「我們 + 模糊」?你的 R 段**有沒有可感知的結果**?
   → 哪段虛,就是你這故事要補的。把它補厚,這就是你的第一張故事卡雛形。

## 本章小結

- 行為面試的全部前提:**過去行為預測未來**。所以你靠的是「事先整理好的真實故事」,不是臨場編。
- **STAR** 講專案 / 成就:**S+T 是引子,A+R 是肉**;死因永遠是 A 太虛、用「我們」、R 沒結果。
- **SBI** 講人 / 衝突:只講**行為 + 影響**,不貼標籤——對事不對人。也是工作中給反饋的標準動作。
- 追問四把鏟子:挖「你」、挖細節、挖反事實、挖衝突點。對策是**主答案就餵足 signal**。
- 行為題就 **6 大類**,備 6-8 個故事全覆蓋。

## 複習自檢

1. STAR 四段哪兩段是「肉」?大多數人最常虛掉哪一段?
2. 為什麼 A 段裡用「我們」是頭號死罪?面試官在過濾什麼?
3. 講「我和同事的摩擦」時,關鍵那句該用 STAR 還是 SBI 的方式說?差別是什麼?
4. 面試官問「如果再來一次你會怎樣」,在考你什麼?
5. 行為題 6 大類各是什麼?你現在手上有故事能 answer 幾類?
